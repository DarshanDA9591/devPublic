# Grand School Website Demo

This is a premium, modern, and fully responsive school website built with Next.js (React + TypeScript), Tailwind CSS, and Framer Motion.

## Features
- Grand, elegant, and visually impressive design
- Mobile-first, fully responsive layout
- Rich color palette (royal blue, maroon, gold accents)
- Smooth scroll-based and entrance animations (Framer Motion)
- Reusable components for all sections:
  - Hero (full-screen banner, CTA)
  - About (vision, mission, values)
  - Infrastructure (animated cards)
  - Academics & Activities (timeline/cards)
  - Gallery (animated grid, lightbox)
  - Chatbot (fixed, modern, sample responses)
  - Footer (address, contact, social, animated)
- Easy to customize text and images
- Optimized for performance

## Getting Started

Run the development server:

```powershell
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Customization
- Edit content and images in the `src/` directory components.
- Update colors and styles in `tailwind.config.js` and `globals.css`.

---

This project is a demo and uses placeholder images/assets. Replace them with your school's branding for production use.
